Landing Page

Este projeto é uma landing page simples criada para simular um site de viagens. O objetivo é apresentar um banner atrativo e seções informativas que fisgam o interesse do visitante.

Introdução

A landing page é uma página inicial pensada para captar a atenção do usuário e incentivá-lo a explorar mais.
Neste projeto, a ideia é remeter a vilarejos europeus ou cidades interessantes ao redor do mundo.

Funcionalidades

- Banner principal com imagem de fundo e texto sobreposto.
- Navegação interna entre seções, sem abrir novas abas.
- Três seções principais:
- TripMe → Planejamento de viagens.
- MeetUs → Conheça a equipe.
- Advice → Dicas práticas de viagem.
- Efeito hover nos links de navegação para melhorar a experiência do usuário.
- Estrutura semântica utilizando <nav>, <main>, <section>, <footer>.
